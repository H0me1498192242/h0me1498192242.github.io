Nessus与Nmap的使用

Nessus是全球使用人数最多的系统漏洞扫描与分析软件，这是一个免费、威力强大、更新频繁并简易使用的远端系统安全扫描程序，功能十分强大。

二、Nessus安装

艹 发现很神奇的现象 我的表情图 好像丢了 还有 我擦 是因为我把输入法卸载了么 麻了 今天早上还没有健康报备 🤮

首先 找个合适的kali版本 然后下载
执行

dpkg -i Nessus-7.1.4-debian6_amd64.deb

安装成功

启动 Nessus

安装完成后出现的这两行，第一行用来启动，第二行用来配置

启动

/bin/systemctl start nessusd

查看nessus 启动状态

netstat -anp


说明启动成功

获取激活码

打开网站：http://www.tenable.com/products/nessus/nessus-plugins/obtain-an-activation-code
选择Free的Register
输入邮箱姓名，邮箱收到消息

登陆 web 界面

https://localhost:8834/


然后注册完 等待下载插件

使用Nessus扫描web

实例演示-扫描漏洞

​


点击“New Scan”新建一个扫描。
选择“Basic Network Scan”，进行配置项目名称，对项目的描述，以及最重要的目标IP地址。
Name随便填，Targets填你要扫得网址，Description可填可不填

在下面输入要扫描的IP
如果我们有目标主机的账号、密码，我们可以点击”Credentials”,进行配置。如果是linux系统就配置SSH，windows就配置windows

5.我们还可以查看我们要用到的插件，点击”Plugins”进行查看。

.全部配置完成之后，我们点击”Save”，进行保存，这样在”My Scan”就能看见我们之前配置过的windows 7。
点击”>”就可以进行扫描，用鼠标点击就能看到详细信息。
我们点击”Vulnerables”，就能看见我们发现的漏洞。

Nmap的使用
nmap是一个网络连接端扫描软件，用来扫描网上电脑开放的网络连接端。确定哪些服务运行在哪些连接端，并且推断计算机运行哪个操作系统。它是渗透测试人员必用的软件之一，以及用以评估网络系统安全。
